Installation
============
* Place files in a web-accessible directory on your web server.
* Create a MySQL compatible database user.
* Visit setup.php in your browser and walk through the screens.

Upgrading
=========
* Extract new files into your existing installation directory.
* Visit upgrade.php in your browser.
